氏名カタカナ推計API
###############################

 カタカナ推計機能は，漢字・アルファベット姓名を入力して，その読みであるカタカナ姓名を出力するものである。 

 本機能は，姓名辞書などを用いず，AIのみを用いた推計である。AIで推計した確率の高い順に出力する。

 漢字姓名と最大カナ姓名返却個数を入力し，カナ姓名の一覧の結果をJSONで取得。最大返却個数は省略可能であり，システムデフォルト値の最大個数５で返す。

APIのURL
===================

 https://api.kktg.digital.go.jp/v1/kana

 * v1
  
  漢字カナ突合モデルのバージョンによってここが変わる

入力パラメタ
====================

 * kanji
  
  漢字姓名を入力。:doc:`data` を参照のこと。

 * max 
  
  最大カナ姓名返却個数。1から10の整数を指定する。


入力例
==============

 * 姓名

  例  https://api.kktg.digital.go.jp/v1/kana?kanji=山田　太郎&max=10

 * 姓のみ

  例 https://api.kktg.digital.go.jp/v1/kana?kanji=山田

 * 名のみ 

  例 https://api.kktg.digital.go.jp/v1/kana?kanji=太郎


出力
============

 出力はHTTPボディにJSON形式で出力される。

 .. code-block:: json

    {
        "response": "OK",  // コード参照
            "result":{
                "status":"OK",
                "kana",[
                    "ヤマダ　タロウ",
                    "ヤマダ　フトロウ",
                       // 最大max個出力  
                ],
                "probability":[
                   0.9996961763137937, // ヤマダ　タロウ の確率
                   4.018219707722746e-10,  // ヤマダ　フトロウ の確率
                   // 最大max個出力  
                ]
            }
        }
    }

出力の項目
=====================

 * repsonse

    :doc:`code` 参照
    
 * status

  + OK
  
   漢字姓名からカナ姓名が得られた。

  + NG
  
   ネットワークエラーなどで漢字姓名からカナ姓名が得られなかった。kanaは0の配列となる
   
 * kana
 
  モデルから取得したカナ姓名一覧。最大max個，ない時（statusがNGの時）は０の配列
