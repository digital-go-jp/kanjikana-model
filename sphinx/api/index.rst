氏名漢字カナ突合API
#########################


氏名漢字カナ突合システムは，利用者登録画面から利用者登録を行い，APIキーを発行することで利用可能となる。
APIキーは複数発行することができる。

利用者登録
============
利用者登録画面にて，利用者の登録及びキーの発行を行い，氏名漢字カナ突合APIで利用するために必要な，キーを発行する。
詳細は  :doc:`./key` を参照のこと。

氏名漢字カナ突合機能
-------------------------
氏名の漢字・アルファベット姓名と，その読みであるカタカナ姓名を入力し，一致しているかどうかを判定する。
詳細は :doc:`./kanji_kana` を参照のこと。


氏名カタカナ推計機能
-------------------------
氏名の漢字・アルファベット姓名を入力し，その読みと推測されるカタカナ姓名を出力する。
詳細は :doc:`./kana` を参照のこと。

氏名漢字推計機能
------------------------
氏名の読みであるカタカナ姓名を入力し，漢字・アルファベット姓名を出力する。
詳細は :doc:`./kanji` を参照のこと。
