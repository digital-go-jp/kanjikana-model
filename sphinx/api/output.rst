出力仕様
###################

出力フォーマット
=======================

 出力はJSON形式で返却する。文字コードはUTF-8で，漢字姓名，カナ姓名に関しては全角文字で返却する。

 出力は，HTTPステータス200で返すこととし，内部エラーや引数エラーの際には，ステータスコードを返却する。

 .. code-block:: json

  {
      "response": "OK",
      "result":{
          "status":99,
             // この部分は機能によって入るものが異なる
      }, 
      "version":"1.6o"
  }
