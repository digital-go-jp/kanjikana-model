データソース
############


氏名漢字カナ突合モデルは，漢字・アルファベットと，その読みであるカタカナのペアのデータを利用して，構築されている。
本モデルで使用したデータソースの一覧は次のとおり




オープンソース辞書データ
====================
* `canna <https://osdn.dl.osdn.net/canna/>`_ 
* `freewnn <https://osdn.dl.osdn.net/freewnn/>`_
* `ipadic <https://taku910.github.io/mecab/>`_
* `kakasi <http://kakasi.namazu.org/>`_
* `mj <https://moji.or.jp/>`_
* `mozc <https://github.com/google/mozc/>`_
* `neologd <(https://github.com/neologd/mecab-ipadic-neologd/>`_
* `skk <https://github.com/skk-dev/dict/>`_

Webデータ
========
