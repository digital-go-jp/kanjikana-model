アルゴリズム

エンジンのアルゴリズムは下記アルゴリズムを用いて漢字とカナのマッチングを行う。また，アルゴリズムを実施する際に，漢字の異体字を置き換える処理や，単語単位でマッチしない場合には一文字単位でマッチするなどを合わせて行っている。

マッチング

  • 辞書マッチングエンジン

    姓名辞書や単漢字辞書を用いて，漢字とカナのペア辞書にあるかどうかで判定する。

  • 外国人マッチングエンジン

    アルファベットとカタカナの組合せパターンから，アルファベットとカナが一致しているか判定する。

  • 統計マッチングエンジン（信頼度高くない）

    統計辞書に漢字とカナのペアがあるかどうかで判定する。本氏名漢字カナ突合モデルでは，統計データを有さないため，使用していない。

  • AIマッチングエンジン（信頼度高くない）

    AIモデルに漢字を入力し，出力された一覧に，カナが入っているかどうかで判断する。
