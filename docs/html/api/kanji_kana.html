氏名漢字カナ突合API

漢字・アルファベット姓名とカタカナ姓名を入力し，姓名辞書，単漢字辞書，異体字辞書や外国人モデル，AIモデルを用いながら，漢字・アルファベットとカタカナ姓名が一致しているかどうかを判定する。

一致の判定モデルは複数用意され，それぞれを選んで使用できる。

漢字姓名とカナ姓名を入力し結果をJSON形式で取得。簡易モデルのため，計算時間は短くレスポンスを優先するが，突合結果の精度が落ちる可能性がある。

APIのURL

  • 簡易モデル

簡易モデルは，単語辞書及び異体字辞書のみで漢字・アルファベット姓名とカタカナ姓名を突合し，漢字・アルファベット姓名とカタカナ姓名が一致しているかどうかを判定し返却する。

https://api.kktg.digital.go.jp/v1/simple

  • 詳細モデル

詳細モデルは，簡易モデルを実施し，一致しないと判定された場合には，

https://api.kktg.digital.go.jp/v1/detail

  • v1

漢字カナ突合モデルのバージョンによってここが変わる

入力

  • kanji

漢字姓名を入力。データ仕様 を参照のこと。

  • kana

全角カタカナで姓名を入力。姓と名の間は全角空白。姓名ではなく，姓のみ，名のみでも突合は可能。その際には，全角空白を使用せず，姓のみもしくは名のみをそれぞれ，kanjiとkanaの入力とすること。

入力例

  • 姓名

例 http://localhost:8080/simple?kanji=山田　太郎&kana=ヤマダ　タロウ

  • 姓のみ

例　http://localhost:8080/simple?kanji=山田&kana=ヤマダ

  • 名のみ

例　http://localhost:8080/simple?kanji=太郎&kana=タロウ

出力

出力はHTTPボディにJSON形式で出力される。

{
    "response": "OK",  // コード参照
        "result":{
            "status":90
        }
    }
}

  • repsonse:

    APIレスポンスコード一覧 参照

  • status

  • 50以上，漢字姓名とカナ姓名が一致

  • 50未満，漢字姓名とカナ姓名が一致していない
