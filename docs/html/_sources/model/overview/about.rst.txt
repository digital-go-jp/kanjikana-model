
漢字カナ突合モデルとは
=====================================

漢字カナ突合モデルとは，漢字姓名（外国人の場合にはアルファベット姓名）とその読み仮名であるカタカナ姓名が一致しているかどうかを判定するモデルである。

一般的に，漢字姓名の読み方は一意に定まらない場合があり，例えば， ``幸子`` と書いて ``ユキコ`` と読む場合や， ``サチコ`` と読む場合など，本人にその読み方を確認しないと，正しい読み方とならない場合がある。

また同様に，アルファベット姓名においても， ``adam`` と書いて ``アダーム`` や ``アダム`` など，その時々の聞こえ方によって，カタカナ表記が異なる場合も多い。

銀行口座の名義人などでは，姓名がカタカナ表記やローマ字表記でなされる時には，戸籍や住民票で記載される漢字姓名が，そのカタカナ表記，ローマ字表記と一致してるかどうかの判定が難しい。

本モデルでは，漢字姓名（アルファベット姓名）と，その読み仮名であるカタカナ姓名を入力とし，漢字姓名の読みがカタカナ姓名と一致しているかどうかを判定する。


漢字姓名（外国人の場合にはアルファベット姓名）とその読み仮名であるカタカナ姓名の一致を判定するモデルである。モデルは大きく分けて，

#. 漢字姓名（アルファベット姓名）とカナ姓名を入力し，一致しているかどうかの判定結果を返す。（:doc:`/overview/kanji_kana/index` ）
#. 漢字姓名（アルファベット姓名）を入力し，カナ姓名の候補一覧を返す。（:doc:`/overview/kanji/index`）
#. カナ姓名を入力し，漢字姓名（アルファベット姓名）の候補一覧を返す。（:doc:`/overview/kana/index`）

の機能を有する。

それぞれの機能は，Dockerイメージとして提供する。URL形式でアクセスし，レスポンスを受け取る。
