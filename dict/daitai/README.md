# 異体字用文字優先順作成用

異体字のペアで、どの文字が最も縮退先として適切かを、JIS水準の低い方を抽出するために、JIS水準の文字を得る。JIS水準が同じ場合には、小学校中学校高校で習う順に縮退先を決める

## JIS水準

別途用意すること
1. [JIS第一水準](./jisdai1.dat)
2. [JIS第二水準](./jisdai2.dat)
3. [JIS第三水準](./jisdai3.dat)
4. [JIS第四水準](./jisdai4.dat)

## 小学校、中学校、高校文字

別途用意すること
1. [小学校](./syogaku.dat)
1. [中学校](./cyugaku.dat)
1. [高校](./koukou.dat)
